# Learning C Using Nothing but ChatGPT (Free)

**"Can someone with little to no programming experience learn C, master Vulkan, and build a rendering engine using only the free version of ChatGPT?"**

*I don't know yet.*  
This repository is my attempt to find out.

## Why This Repository Exists

I wanted to see if it's possible.

Can a complete beginner — someone with almost zero programming experience — actually learn C properly, then go all the way to mastering Vulkan and building their own rendering engine, using **nothing but the free version of ChatGPT**?

That’s the question I’m trying to answer.

No paid AI.  
No courses.  
No YouTube playlists.  
No mentor.  
No structured roadmap.

Just me, ChatGPT (free), and curiosity.

## The Real Process

There was no plan.

I just started chatting with ChatGPT.

I went from `printf` to `scanf`, then to the `&` symbol, then to pointers… and somehow ended up learning pointers before properly understanding variables.

It’s been messy, random, and full of mistakes — but that’s exactly how I want to document it.

## My Goal

This is not just about learning C.

The end goal is much bigger:
- Master C
- Learn the mathematics behind computer graphics
- Master the Vulkan Graphics API
- Build my own rendering engine from scratch

I’m choosing the hard low-level path because I want to understand what the computer is really doing.

## What You’ll Find Here

This repo is a raw diary of the journey.

You will see:
- Tiny experiments
- Practice programs
- Lots of mistakes and bad code
- Random files made just to test one idea
- Gradual improvement over time

Everything — including the ugly parts — stays here.

## Final Message

This is one person’s real attempt to prove (or disprove) that you can go from almost zero knowledge all the way to building a rendering engine using only free ChatGPT.

I don’t know if I’ll succeed.

But every question, every mistake, every small breakthrough, and every line of code will be documented here.

Let’s see how far this goes.
